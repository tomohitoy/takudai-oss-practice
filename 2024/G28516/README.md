# 課題「ビジネスアイデア」

### 興味を持っているビジネス領域・業界
- 教育業（初等，中等，高等，職業，語学）


### 上記のビジネス領域で注目する課題
- 教育格差の拡大。家庭環境や地域による教育機会の差が拡大し、教育格差が生まれる。
- 経済的な理由などで、質の高い教育を受けられない子どもたちが増える。

### 上記のビジネス領域で提案する解決策
- 低所得家庭の子どもたちへの学習支援や、地域における教育資源の共有を促進させる。
- オンライン学習の活用により、時間や場所の制約なく、質の高い教育を提供させる。
